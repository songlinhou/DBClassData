# DB Helper -- A great tool for DB class!

## How to Use?

First time use.

```
$ git clone https://github.com/songlinhou/DBClassData

$ cd DBClassData/

$ chmod +x init.sh

$ ./init.sh

SQL> @createTables;

SQL> exit;

```

Quick Start.

```
$ ./db.sh 
```

Change Your Username.

```
$ rm .name
$ ./db.sh
```
